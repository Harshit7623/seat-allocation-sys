# DB Visualizer - Frontend

Interactive database schema visualizer built with React and React Flow.

## Installation

```bash
cd frontend
npm install
```

## Development

```bash
npm run dev
```

The application will open at `http://localhost:3000`

## Build

```bash
npm run build
```
